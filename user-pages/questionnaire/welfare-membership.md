# Welfare Membership Questionnaire — Usage

1. Visit `/questionnaires/welfare-membership` or pick **Welfare Membership** from the navigation bar.
2. Complete the member details fields, noting that mandatory prompts show a red asterisk.
3. Record dependants with the **Add another** button in the children section; remove extras with **Remove** when more than the minimum entry exists.
4. Submit the form. While the submission is in progress the primary button reads “Submitting…”.
5. Watch for the confirmation banner signalling success or the error banner instructing the visitor to try again later.
